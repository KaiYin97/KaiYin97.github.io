# Overlay for AcademicPages (Kai Yin)

This folder contains content files customized from your CV.

How to use:
1) Create your website repo using the AcademicPages template.
2) Copy the folders in this overlay into the root of your repo (overwrite when prompted).
3) Commit + push. GitHub Pages will rebuild automatically.

Notes:
- Add your headshot at `images/profile.png` (or update the `avatar` path in `_config.yml`).
- Update `_config.yml` with your site URL and social links.
